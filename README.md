# Getting started

## install git
* 윈도우 환경에서는 git for windows와 CUI 환경이 싫다면 tortoisegit을 까는 것을 추천함
* 다른 환경은 나도 몰라
* ssh를 이용한 authencication이 필요하다면 ssh 클라이언트는 OpenSSH를 사용하는것을 추천함
